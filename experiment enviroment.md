一、實驗系統架構（軟體部分）

本研開發之「線上英文單字學習與測驗系統 (cool-vocabulary-learning-app-v2)」作為主要實驗工具，其軟體環境配置如下：

- 開發語言與框架：系統採用 **Python** 語言開發，並結合 **FastAPI** 網頁框架進行非同步邏輯處理與 API 管理。
- 使用者認證系統：整合 **Google OAuth 2.0** 服務，確保受試者身份登入之安全性與唯一性，並簡化實驗註冊流程。
- 資料庫管理系統：使用 **SQLite** 關係型資料庫儲存受試者之學習歷程、測驗成績及作答反應時間等實驗數據。
- 前端介面：採用 HTML5、CSS3 及 JavaScript 構建響應式網頁介面（Responsive Web Design），確保受試者在不同裝置上皆具備一致的學習體驗。

二、實驗硬體設備

實驗過程涉及伺服器端與受試者端之裝置：

- 伺服器端 (Server Side)：
    - 系統部署於 **Render** 雲端平台。
    - 規格配置：**RAM 512MB, vCPU 1** 以上，以支援多位受試者同時在線進行學習與測驗。
- 受試者端 (Client Side)：
    - 裝置類型：受試者可使用桌上型電腦、筆記型電腦、平板或智慧型手機參與實驗。
    - 輸入設備：標準鍵盤或螢幕觸控鍵盤。
    - 瀏覽器要求：需支援最新版本之主流瀏覽器（如 Google Chrome, Safari, Microsoft Edge）。

### **三、實驗材料**

- 單字庫：內建針對實驗設計之英文單字庫，包含單字定義、例句及多媒體素材。
- 測驗試卷：系統會自動根據受試者分組，按固定順序呈現題目。
- 數據採集腳本：後端內建自動化數據收集工具，用於即時記錄受試者的學習效率與正確率，以便後續進行統計分析。

### **四、網路環境**

- 實驗全程需在穩定之網際網路連接環境下進行，以利與資料庫進行即時資料同步。
